Corporate Document Intelligent System (CDIS)OverviewThe Corporate Document Intelligent System (CDIS) is an end-to-end automated framework designed to transform the corporate data lifecycle. By integrating an advanced ETL pipeline with Natural Language Processing (NLP), CDIS extracts both quantitative financial Key Performance Indicators (KPIs) and qualitative non-financial narratives from diverse documents like earnings reports and press releases.The system bridges the gap between massive volumes of unstructured data and actionable business intelligence, providing a 360-degree view of corporate performance.Key FeaturesAutomated Data Ingestion: Targeted web scraping of financial disclosures and Google RSS feeds for real-time news acquisition.Hybrid Extraction Architecture: A multi-pass system utilizing deterministic Regex, Fuzzy Semantic Mapping, and Gemini 2.5 for high-fidelity long-context reasoning.Predictive Analytics: Forecasting module supporting ARIMA, Prophet, and LSTM models to predict future financial trends and guidance deviations.Intelligent Chatbot: A natural language interface for executing complex cross-company and cross-time queries.Comprehensive Visualization: Interactive Power BI dashboards and Python-based visual analytics (Matplotlib/Plotly).System Architecture1. Data Collection & PreprocessingSources: Investor Relations portals (PDFs) and corporate news (HTML).OCR Integration: Convolution-based recognition models for scanned or image-based files.Normalization: Unicode harmonization, boilerplate removal, and tokenization (Byte-Pair Encoding).2. Information Extraction (Technical & Non-Technical)Technical: Extracts Revenue, EPS, EBITDA, and Guidance ranges with evidence-backed spans for auditability.Non-Technical: Processes news for sentiment polarity, event extraction (Actor, Action, Object), and summarization.3.
